The Tony Santana Portfolio includes automatic vCard generation at the `/vcard` endpoint, allowing visitors to easily add your contact information to their address books.

## Overview

vCards (Virtual Contact Files) are standardized digital business cards that can be imported into contact management applications. The portfolio automatically generates a vCard from your profile information with support for photos, contact details, and job information.

## Accessing the vCard

The vCard is available at:

```
https://yoursite.com/vcard
```

Visiting this URL automatically downloads a `contact.vcf` file that can be opened in:

- iOS Contacts
- Android Contacts
- Gmail
- Outlook
- Apple Mail
- Other contact management apps

## Implementation

The vCard generation is implemented in `src/app/vcard/route.ts:9`:

```typescript
import { NextResponse } from "next/server";
import sharp from "sharp";
import VCard from "vcard-creator";
import { PROFILE } from "@/content/profile";
import { decodeEmail, decodePhoneNumber } from "@/lib/string";
import { getLastCompany } from "@/services/experience";

export async function GET() {
  const card = new VCard();

  card
    .addName(PROFILE.lastName, PROFILE.firstName)
    .addPhoneNumber(decodePhoneNumber(PROFILE.phoneNumber))
    .addAddress(PROFILE.address)
    .addEmail(decodeEmail(PROFILE.email))
    .addURL(PROFILE.website);

  const photo = await getVCardPhoto(PROFILE.avatar);
  if (photo) {
    card.addPhoto(photo.image, photo.mine);
  }

  const lastCompany = getLastCompany(PROFILE.sections.experiences?.items ?? []);

  if (lastCompany) {
    card.addCompany(lastCompany.companyName)
        .addJobtitle(lastCompany.positions[0].title);
  }

  return new NextResponse(card.toString(), {
    status: 200,
    headers: {
      "Content-Type": "text/vcard; charset=utf-8",
      "Content-Disposition": "inline; filename=contact.vcf",
      "Cache-Control": "public, max-age=86400, s-maxage=86400",
      "X-Robots-Tag": "noindex, nofollow",
    },
  });
}
```

## Contact Information Encoding

Email addresses and phone numbers are stored in encoded format to prevent spam bots from harvesting contact information.

### Email Encoding

Emails are base64 encoded in the profile:

```typescript
import { decodeEmail } from "@/lib/string";

// Profile stores encoded email
const encodedEmail = "dG9ueXNhbnRhbmExNDkyQGdtYWlsLmNvbQ==";

// Decode when needed
const email = decodeEmail(encodedEmail);
// Returns: "tonysantana1492@gmail.com"
```

### Phone Number Encoding

Phone numbers are also encoded:

```typescript
import { decodePhoneNumber } from "@/lib/string";

// Profile stores encoded phone
const encodedPhone = "KzE3ODY3MzU3NzQ5";

// Decode when needed
const phone = decodePhoneNumber(encodedPhone);
// Returns: "+17867357749"
```

<Tip>
  This encoding prevents automated bots from scraping your contact information while keeping it accessible to legitimate users via vCard download.
</Tip>

## Photo Processing

Profile photos are automatically converted to JPEG format and optimized for vCard compatibility:

```typescript
async function getVCardPhoto(url: string) {
  try {
    const res = await fetch(url);

    if (!res.ok) {
      return null;
    }

    const buffer = Buffer.from(await res.arrayBuffer());
    if (buffer.length === 0) {
      return null;
    }

    const contentType = res.headers.get("Content-Type") || "";
    if (!contentType.startsWith("image/")) {
      return null;
    }

    const jpegBuffer = await convertImageToJpeg(buffer);
    const image = jpegBuffer.toString("base64");

    return {
      image,
      mine: "jpeg",
    };
  } catch {
    return null;
  }
}

async function convertImageToJpeg(imageBuffer: Buffer): Promise<Buffer> {
  try {
    const jpegBuffer = await sharp(imageBuffer)
      .jpeg({
        quality: 90,
        progressive: true,
        mozjpeg: true,
      })
      .toBuffer();

    return jpegBuffer;
  } catch (error) {
    console.error("Error converting image to JPEG:", error);
    throw error;
  }
}
```

### Image Conversion Features

- Fetches avatar from URL
- Validates image format
- Converts to JPEG using Sharp
- Optimizes with 90% quality
- Uses progressive JPEG encoding
- Base64 encodes for vCard format

## Response Headers

The vCard endpoint returns specific headers for proper handling:

```typescript
headers: {
  "Content-Type": "text/vcard; charset=utf-8",
  "Content-Disposition": "inline; filename=contact.vcf",
  "Cache-Control": "public, max-age=86400, s-maxage=86400",
  "X-Robots-Tag": "noindex, nofollow",
}
```

<Accordion title="Content-Type">
  `text/vcard; charset=utf-8` - Identifies the file as a vCard with UTF-8 encoding
</Accordion>

<Accordion title="Content-Disposition">
  `inline; filename=contact.vcf` - Suggests the filename when downloaded
</Accordion>

<Accordion title="Cache-Control">
  `public, max-age=86400` - Caches the vCard for 24 hours (86400 seconds)
</Accordion>

<Accordion title="X-Robots-Tag">
  `noindex, nofollow` - Prevents search engines from indexing the vCard endpoint
</Accordion>

## Updating Contact Information

To update the vCard data, edit your profile configuration:

1. Update profile data in `src/content/profile.ts`
2. Encode new email/phone if changed
3. Update avatar URL if changed
4. Update work experience for job title/company

The vCard will automatically reflect these changes.
