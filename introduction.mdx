  <h2 className="text-2xl sm:text-3xl font-semibold text-gray-900 dark:text-white mb-4">
    Quick Start
  </h2>
  <p className="text-base text-gray-600 dark:text-gray-400 mb-8">
    Get your portfolio up and running in minutes
  </p>

  <Steps>
    <Step title="Clone the repository">
      Clone the repository to your local machine using Git:
      
      ```bash
      git clone https://github.com/tonysantana1492/portfolio.git
      cd portfolio
      ```
    </Step>

    <Step title="Install dependencies">
      Install the required dependencies using pnpm (recommended), npm, or yarn:
      
      <CodeGroup>
      ```bash pnpm
      pnpm install
      ```
      
      ```bash npm
      npm install
      ```
      
      ```bash yarn
      yarn install
      ```
      </CodeGroup>
    </Step>

    <Step title="Configure your profile">
      Update the profile data in `src/content/profile.ts` with your own information:
      
      ```typescript
      export const PROFILE = {
        displayName: "Your Name",
        username: "yourusername",
        bio: "Your bio goes here",
        // ... more configuration
      }
      ```
      
      <Tip>See the [Profile Data](/customization/profile-data) guide for detailed configuration options.</Tip>
    </Step>

    <Step title="Start the development server">
      Launch the development server and view your portfolio:
      
      ```bash
      pnpm dev
      ```
      
      Your portfolio will be available at `http://localhost:3000`
    </Step>
  </Steps>
</div>
