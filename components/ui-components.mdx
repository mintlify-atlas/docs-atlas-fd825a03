## Button

A versatile button component with multiple variants and sizes. Located at `src/components/ui/button.tsx:45`.

### Usage

```tsx
import { Button } from "@/components/ui/button"

<Button variant="default" size="default">
  Click me
</Button>
```

### Variants

<Tabs>
  <Tab title="Visual Styles">
    - `default` - Primary button with gradient background
    - `secondary` - Secondary styling with muted colors
    - `destructive` - Red destructive actions
    - `outline` - Bordered transparent background
    - `ghost` - No background, hover effect only
    - `link` - Underlined link styling
  </Tab>
  <Tab title="Sizes">
    - `default` - Standard height (h-8)
    - `sm` - Small size (h-7)
    - `lg` - Large size (h-10)
    - `icon` - Square icon button (size-8)
    - `icon:sm` - Small icon button (size-7)
    - `icon:lg` - Large icon button (size-10)
  </Tab>
</Tabs>

### Props

```typescript
type ButtonProps = React.ComponentProps<"button"> & {
  variant?: "default" | "secondary" | "destructive" | "outline" | "ghost" | "link"
  size?: "default" | "sm" | "lg" | "icon" | "icon:sm" | "icon:lg"
  asChild?: boolean
}
```

### Advanced Usage

The `asChild` prop allows you to render the button as any element:

```tsx
import { Button } from "@/components/ui/button"
import Link from "next/link"

<Button asChild variant="outline">
  <Link href="/about">About</Link>
</Button>
```

## Dialog

Modal dialog component built on Radix Dialog primitive. Located at `src/components/ui/dialog.tsx:9`.

### Usage

```tsx
import {
  Dialog,
  DialogContent,
  DialogDescription,
  DialogHeader,
  DialogTitle,
  DialogTrigger,
} from "@/components/ui/dialog"

<Dialog>
  <DialogTrigger asChild>
    <Button>Open Dialog</Button>
  </DialogTrigger>
  <DialogContent>
    <DialogHeader>
      <DialogTitle>Dialog Title</DialogTitle>
      <DialogDescription>
        This is a description of the dialog content.
      </DialogDescription>
    </DialogHeader>
    <p>Dialog content goes here.</p>
  </DialogContent>
</Dialog>
```

### Features

- Accessible modal with proper ARIA attributes
- Backdrop overlay with blur effect
- Close button (can be hidden with `showCloseButton={false}`)
- Smooth open/close animations
- Escape key and outside click to close

## Styling Best Practices

### Using the cn() Utility

Always use `cn()` to combine class names:

```tsx
import { cn } from "@/lib/utils"

function CustomButton({ className, ...props }) {
  return (
    <button
      className={cn(
        "rounded-md px-4 py-2",  // Base styles
        "hover:bg-accent",        // State styles
        className                  // Override prop
      )}
      {...props}
    />
  )
}
```

### Component Composition

Build complex UIs by composing simple components:

```tsx
import { Button } from "@/components/ui/button"
import { Input } from "@/components/ui/input"
import { Panel, PanelContent } from "@/components/ui/panel"

<Panel>
  <PanelContent>
    <form className="flex gap-2">
      <Input placeholder="Enter text" />
      <Button type="submit">Submit</Button>
    </form>
  </PanelContent>
</Panel>
```
